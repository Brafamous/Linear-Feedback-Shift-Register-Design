============================================================
SECTION 1 — INTRODUCTION TO LFSR
============================================================

Definition:
-----------
An LFSR (Linear Feedback Shift Register) is a shift register
whose input bit is generated using XOR feedback logic from
selected register bits called taps.

Applications:
-------------
- Pseudo-random number generation
- Built-In Self-Test (BIST)
- Cryptography
- Communication systems
- Error detection
- ASIC/FPGA verification

Important Concepts:
-------------------
1. Pseudo-random sequences
2. XOR feedback
3. Sequential logic
4. Shift registers
5. Primitive polynomials
6. Tap selection

SECTION 4 — FEEDBACK LOGIC
============================================================

Feedback Equation:
------------------
feedback = lfsr_reg[7] ^
           lfsr_reg[5] ^
           lfsr_reg[4] ^
           lfsr_reg[3];

Explanation:
------------
These are selected tap positions.

Primitive Polynomial:
---------------------
x^8 + x^6 + x^5 + x^4 + 1

Tap Mapping:
------------
x^8 -> bit7
x^6 -> bit5
x^5 -> bit4
x^4 -> bit3

Purpose:
--------
Generate maximal-length pseudo-random sequence.

============================================================
SECTION 5 — SHIFT OPERATION
============================================================

Shift Statement:
----------------
lfsr_reg <= {feedback, lfsr_reg[7:1]};

Explanation:
------------
1. feedback inserted at MSB
2. Existing bits shift right
3. LSB discarded

Concatenation Operator:
-----------------------
{A,B} joins signals together.

SECTION 8 — GENERATION LOGIC
============================================================

Condition:
----------

Generation Operation:

SECTION 9 — SERIAL OUTPUT LOGIC
============================================================

Condition:
----------

Operation:
----------

Shift:
------

Output Order:
-------------

Example:
--------

Output Sequence:
----------------
